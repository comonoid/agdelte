Dynamic Optics

ixList + Traversal: runtime optic composition and batch operations

How it works: Click an item to select it. Use +1 Selected / −1 Selected to modify the selected item's value via a runtime-composed optic (ixList n ∘O itemsLens). Reset All sets every item to 0 using a Traversal. + Add Item prepends a new item. The info panel shows the current selection (via peek — safe, may return nothing if index is out of bounds).
