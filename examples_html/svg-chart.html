SVG Bar Chart

Data visualization with foreach, reactive bindings, and click selection

Expected: An SVG bar chart with colored bars. Click a bar to select it (it turns dark blue). Buttons +/− adjust the selected bar's value. Add appends a new bar, Remove deletes the selected one. Bar heights, labels, and colors update reactively. A Y-axis with gridlines is drawn. The selected bar info is displayed below the chart.
