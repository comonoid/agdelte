SVG Pan/Zoom

Interactive pan/zoom with SVG events and coordinate conversion

Expected: An SVG canvas with colored shapes. Click-and-drag to pan the viewport. Scroll wheel (or +/− buttons) to zoom in/out. A Reset button returns to the default view. Coordinates are converted from screen to SVG space via getScreenCTM().inverse(). The current viewBox values are displayed reactively.
