WebSocket Demo

Real-time echo client with WebSocket

Expected: A Connect button and a text input. After connecting to the local echo server (ws://localhost:8080/echo, built into serve.cjs), type a message and click Send. The sent message appears on the right (blue, "You"), the echoed response on the left (green, "Echo") — the same text back. Status shows connected/disconnected/error. Disconnect closes the WebSocket.
