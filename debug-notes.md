# Заметки по темам отладки

Рекомендации, наблюдения и идеи, возникающие при работе с темами из `debug-plan-all.md`.
Не баги — а вещи которые стоит запомнить: недостающие тесты, неточные комментарии,
возможные улучшения, подозрительные места для повторного анализа.

Формат: секция на тему (по ID из debug-plan-all.md), добавляется по мере работы.
